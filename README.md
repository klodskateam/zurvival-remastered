# Zurvival Remastered

![логотип Zurvival Remastered](./Resources/Extra/ZR.png)
Zurvival Remastered - игра, в которой вы попали в зомби-апокалипсисе, и вам нужно отстреливаться от зомби, чтобы выжить.

Текущая версия: **2.0**

$~$
## Моды
![логотип Zurvival Remastered](./Resources/Extra/mods-ru.png)

$~$

В игре есть два типа модов: **Кор-моды** и **json-моды**.

**Кор-моды** это классический тип модов путём изменения кода в исходниках. Эти моды напрямую изменяют игру. На одну сборку игры может быть только один Кор-мод, и каждая модифицированная сборка игры считается Кор-модом.

**Json-моды** это новый тип модов, который позволяет делать *некоторые* модификации в игре без изменения исходников. Эти моды обычно распространяются в zip-архиве, в которых есть сам json-файл и его ресурсы.

В отличии от Кор-модов, для json-модов не требуется скачивать новую сборку игры (за исключением если указано иначе, или json-мод был сделан для более новой версии ZR)

Json-моды ставятся в папку (на Windows) ``%appdata%\Godot\app_userdata\Zurvival Remastered\mods``

## Поддержите разработку
![логотип Zurvival Remastered](./Resources/Extra/supportdevelopment-ru.png)

Для исправления багов или при желании улучшить базовую игру не обязательно делать свой форк игры. Можно открыть Pull Request если умеете работать в Godot или Issue в случае если вы не умеете работать в Godot, чтобы помочь разработке.
